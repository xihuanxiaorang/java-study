## Spring 版本命名规则

| 描述方式 | 说明     | 含义                                     |
| -------- | -------- | ---------------------------------------- |
| Snapshot | 快照版   | 尚不稳定、处于开发种的版本               |
| Release  | 稳定版   | 功能相对稳定，可以对外发行，但有时间限制 |
| **GA**       | **正式版**   | 代表广泛可用的稳定版                     |
| M        | 里程碑版 | 具有一些全新的功能或具有里程碑意义的版本 |
| RC       | 终测版   | 最终测试，即将作为正式版发布                                         |
